#  E-Commerce Sales Dashboard (Excel)

## Project Overview

This project was completed as part of a Data Analytics course. The objective was to design an interactive Excel dashboard that analyzes E-commerce sales data and provides business insights through charts and user controls.

## Features

- Month-wise Sales and Profit analysis
- Region-wise Sales analysis
- Product Category filter using Combo Box
- Interactive column charts

## Dataset

The dataset contains E-commerce sales information, including:

- Order ID
- Order Date
- Ship Date
- Product Category
- Product
- Sales
- Quantity
- Discount
- Profit
- Shipping Cost
- Customer Details
- Region
- Month

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Combo Box (Form Control)
- Data Analysis Add-in
